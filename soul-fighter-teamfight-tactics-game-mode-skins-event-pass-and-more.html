Soul Fighter Teamfight Tactics: Game Mode, Skins, Event Pass And More

Teamfight Tactics is set to get a brand new in-game event — Soul Fighter. Here's everything that's coming to the auto-battler when the event launches.

First Published June 26, 2023, 15:00

Riot Games has announced a brand new event heading to all four of the Runeterra games. Soul Fighter is a six-week event that will roll out across League of Legends, Teamfight Tactics, Wild Rift, and Legends of Runeterra. According to Riot Games, the event is a tribute to the fighting game genre, and takes inspiration from unique character moments from fighting games, anime and more. 

As part of the event, players can expect in-game experiences, new game modes, and cosmetics across all of the four titles. Soul Fighter takes place in an alternate universe, where competitors prove themselves through combat by taking part in the Tournament of Souls. In Teamfight Tactics, players will have not only a new game mode to try out, but a narrative experience exclusive to TFT. 

Soul Fighter Teamfight Tactics: Soul Brawl Game Mode Explained

As part of the Soul Fighter event, players will be able to take part in Soul Brawl — a new, limited-time combat-focused game mode. In Soul Brawl, players will be entering two stages; Training phase, where players compete for Soul Power to build Soul Crown and earn additional loot options, and Tournament phase, where players will be eliminated in a best-of-three bracket tournament. 

Soul Fighter Teamfight Tactics: The Choncc Dome

Exclusive to TFT is The Choncc Dome. This in-client narrative experience sees Champions and Tacticians team up in preparations for the Tournament of Souls. This is the first time a TFT in-client experience has been brought to mobile platforms. 

Soul Fighter Teamfight Tactics: Tactician Cosmetics Revealed

Two new Tactician cosmetics will be heading to TFT as part of the Soul Fighter event. This time for Gwen. Players will be able to get their hands on the following:

  • Chibi Gwen
  • Chibi Soul Fighter Gwen (Mythic)

In addition to skins for Gwen, players will also be able to grab an Arena cosmetic, this time based on the Tournament of Souls. If that wasn't enough, the Soul Fighter Event Pass will also open up new rewards for players to get their hands on.

Soul Fighter Teamfight Tactics: Start, End Date

The Soul Fighter event start globally on Thursday, July 20, 2023. The event will come to a close on Monday, August 28, spanning six weeks. Soul Fighter will go live across all four Runeterra titles: League of Legends, Teamfight Tactics, Wild Rift, and Legends of Runeterra.
